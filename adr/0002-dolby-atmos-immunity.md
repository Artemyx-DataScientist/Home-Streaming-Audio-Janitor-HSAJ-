# ADR 0002 — Иммунитет к Dolby Atmos

- Статус: Принят
- Контекст: Платформы могут помечать треки как Dolby Atmos, но наша доменная логика должна корректно обрабатывать такие варианты без деградации качества данных.
- Решение: Рассматривать флаг Atmos как устойчивый атрибут, не требующий специальных деградаций при обработке. Core должен сохранять/передавать этот статус неизменённым.
- Последствия: Миграции/нормализация обязаны сохранять Atmos-флаг; тестовые сценарии должны покрывать сохранение этого атрибута; дальнейшие правила качества не должны «понижать» Atmos без явного требования.
